# Bill of Materials

| Reference | Component                | Value / Part         | Package / Footprint Used                           | Purpose / Notes                                                                                                                                                                               |
| --------- | ------------------------ | -------------------- | -------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| U1        | Dual op-amp / comparator | LM358                | DIP-8                                              | Used as the comparator. One op-amp section compares the thermistor voltage with the adjustable threshold voltage. The unused second op-amp section is tied off properly so it does not float. |
| Q1        | N-channel MOSFET         | IRLIZ44N             | TO-220-3 vertical                                  | Low-side switch for the 12V fan. The MOSFET handles the fan current while the LM358 only drives the gate.                                                                                     |
| RT1       | NTC thermistor           | 10k @ 25°C           | Through-hole thermistor / resistor-style footprint | Temperature sensor. Its resistance decreases as temperature rises, changing the `TEMP_SENSE` voltage.                                                                                         |
| RV1       | Potentiometer            | 10k                  | Bourns 3386P vertical trimmer footprint            | Sets the adjustable temperature threshold. Turning the potentiometer changes the `REF_SET` voltage going into the LM358.                                                                      |
| R1        | Resistor                 | 10k                  | Axial through-hole resistor                        | Forms the lower half of the thermistor voltage divider with RT1.                                                                                                                              |
| R2        | Resistor                 | 100k                 | Axial through-hole resistor                        | Hysteresis feedback resistor from the LM358 output back to the temperature sense input. Helps prevent fan chatter near the switching point.                                                   |
| R3        | Resistor                 | 100Ω                 | Axial through-hole resistor                        | MOSFET gate resistor. Limits sudden gate charging current and helps keep the gate drive stable.                                                                                               |
| R4        | Resistor                 | 100k                 | Axial through-hole resistor                        | MOSFET gate pulldown resistor. Keeps the MOSFET off when the comparator output is inactive or during startup.                                                                                 |
| R5        | Resistor                 | 2.2k                 | Axial through-hole resistor                        | Current-limiting resistor for the power indicator LED.                                                                                                                                        |
| D1        | Rectifier diode          | 1N4007               | DO-41 through-hole diode                           | Flyback diode across the fan output. Protects the MOSFET from voltage spikes when the fan motor switches off.                                                                                 |
| D2        | LED                      | Power indicator LED  | 5mm through-hole LED                               | Turns on when the board receives power. Used as a simple power status indicator.                                                                                                              |
| C1        | Ceramic capacitor        | 100nF                | Through-hole disc capacitor                        | LM358 decoupling capacitor. Helps reduce small supply noise close to the op-amp power pins.                                                                                                   |
| C2        | Electrolytic capacitor   | 10µF                 | Radial through-hole capacitor                      | Bulk input capacitor across the 12V supply. Helps stabilize the input rail when the fan switches.                                                                                             |
| J1        | Screw terminal connector | 2-pin terminal block | 5.08mm pitch through-hole terminal block           | 12V DC power input connector. Pin 1 is +12V and pin 2 is GND.                                                                                                                                 |
| J2        | Screw terminal connector | 2-pin terminal block | 5.08mm pitch through-hole terminal block           | Fan output connector. Pin 1 connects to +12V and pin 2 connects to `FAN_LOW`, which is switched by the MOSFET.                                                                                |
